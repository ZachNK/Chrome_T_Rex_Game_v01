# Chrome_T_Rex_Game_v01

개선할 점:
1. 모바일 규격에 적절한 화면 비율 및 고정 기능 필요.
2. 장애물에 닿을 때가 아닌, 장애물을 넘은 후 점수 획득하는 방식으로 할 것.
3. 난이도별 스테이지 업그레이드 등 추가할 것.
